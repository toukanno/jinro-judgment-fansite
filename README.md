# 人狼ジャッジメント攻略ファンサイト

人狼ジャッジメントの役職情報、立ち回り、初心者向け解説、編成考察をまとめるファンサイトです。

## 技術スタック

- **Next.js** (App Router / Static Export)
- **TypeScript**
- **Tailwind CSS v4**

## 開発

```bash
npm install
npm run dev
```

## ビルド

```bash
npm run build
```

静的ファイルが `out/` ディレクトリに生成されます。

## ページ構成

- `/` - トップページ
- `/roles` - 役職一覧
- `/guide` - 初心者ガイド
