# Jinro Judgment Fan Site

人狼ジャッジメントの役職情報、立ち回り、初心者向け解説、編成考察をまとめるファンサイトです。

## セットアップ

```bash
npm install
```

## 開発

```bash
npm start
```

Electron ウィンドウでアプリが起動します。

## Windows 向けビルド

```bash
npm run build:win
```

`release/` ディレクトリにインストーラー (.exe) が生成されます。

ポータブル版をビルドする場合:

```bash
npm run build:win:portable
```

## 機能

- 役職一覧
- 初心者ガイド
- 立ち回りガイド
- 編成考察
